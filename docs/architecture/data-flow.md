## Data Classification

| Data Type | Classification | Storage | Encryption | Retention |
|-----------|---------------|---------|------------|-----------|
| User credentials | Confidential | Vault | AES-256 | Session only |
| Audit logs | Internal | PostgreSQL + S3 | At-rest (KMS) | 7 years |
| Service configs | Internal | Git + PostgreSQL | At-rest | Indefinite |
| Deployment artifacts | Internal | ECR + S3 | At-rest | 90 days |
| Metrics data | Internal | Prometheus/Thanos | None | 30 days |
| PII (user profiles) | Restricted | PostgreSQL | AES-256 (KMS) | Account lifetime |
